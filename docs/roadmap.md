# Roadmap

## Nächster sinnvoller Ausbau

1. SSML sauber modellieren
   - Sprachwechsel
   - Pausen
   - Betonung
   - Prosodie

2. Viewer robuster machen
   - Caching
   - Vorladen der nächsten Folie
   - besseres Error Handling für Remote-Ressourcen

3. Format vereinheitlichen
   - eindeutiges Manifest-Schema
   - Validierung beim Laden
   - optionale Versionierung des Manifests

4. Autoring ergänzen
   - ZIP-Erzeugung im Browser
   - neues Folienset anlegen
   - Medien komfortabel hinzufügen

5. Packaging
   - PWA-Modus
   - Offline-Nutzung
   - Desktop-Wrapping bei Bedarf
